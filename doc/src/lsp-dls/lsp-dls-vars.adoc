[id="lsp-dls-vars"]
==== `lsp-dls` settings

[id="lsp-dls-path"]
- `lsp-dls-path`
____
Default value: `pass:[("dls")]`

Path to dls server binary.

NOTE: Introduced in `lsp-mode` 6.3
____
[id="lsp-dls-args"]
- `lsp-dls-args`
____
Default value: `pass:[nil]`

Add server initialization options for dls.

NOTE: Introduced in `lsp-mode` 6.3
____
